void CraigFowler.Web.ZPT.Tal.TalOutput.WriteEndElement ( TalContent  content)

Writes the end of an XML element and a trailing newline character if the configuration of this instance deems that this is appropriate.

This method is responsible for writing the end element itself as well as inserting a new line character. It is not responsible for any indentation. The indentation must be performed before and after this method is used (as appropriate).

Parameters:
contentA TalContent that describes the node to be written.

void CraigFowler.Web.ZPT.Tal.TalOutput.WriteText ( TalContent  content)

Writes text content to the output. This method might also write leading indentation and a trailing newline if the configuration of this instance deems that these are appropriate.

When writing content, if PerformReformatting is enabled then the content will be trimmed of leading and trailing whitespace before being written to the output stream.
