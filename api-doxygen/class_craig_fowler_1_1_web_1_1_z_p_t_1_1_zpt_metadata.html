static ZptMetadata CraigFowler.Web.ZPT.ZptMetadata.GetMetadata ( FileInfo  documentFile) [static]

Overloaded. Gets an instance of ZptMetadata for a ZPT template document.

In any non-exception scenario this method will always return a non-null ZptMetadata instance. If the template file has no explicit metadata defined then a default metadata instance will be created for that template.

Parameters:
documentFileA FileInfo that describes the location of the ZPT document file. The metadata file will be automatically detected based on its extension.
See also:
