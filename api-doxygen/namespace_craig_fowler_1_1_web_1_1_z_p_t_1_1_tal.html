enum CraigFowler::Web::ZPT::Tal::TalContentType

Enumerates the types of TAL content that can be written when rendering a TalElement.

Enumerator:
Text 

Indicates that the content should be written directly as text, thus all special XML characters should be escaped.

Structure 

Indicates that the content should be writen verbatim, retaining any embedded XML structure.
