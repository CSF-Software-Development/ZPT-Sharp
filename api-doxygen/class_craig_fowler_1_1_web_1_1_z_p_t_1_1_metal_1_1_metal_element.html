Dictionary<string,MetalElement> CraigFowler.Web.ZPT.Metal.MetalElement.GetFilledSlots ( )

Gets a collection of the child elements of the current element that contain METAL 'fill-slot' attributes, indexed by the value of the name of the slot that they fill.

Returns:
A dictionary of System.String and MetalElement.

If GetUseMacro(Tales.TalesContext) returns a non-null MetalMacro instance then this method will always return a non-null collection, although it may be empty (if the use-macro directive contains no fill-slot directives).

Exceptions:
MetalExceptionIf GetUseMacro(Tales.TalesContext) returns a null reference (indicating that this element instance does not contain a 'use-macro' directive).
virtual MetalMacro CraigFowler.Web.ZPT.Metal.MetalElement.GetUseMacro ( Tales.TalesContext  context,
bool  bypassCache 
) [virtual]

Overloaded. Gets a MetalMacro instance for which to use with this element.

This method provides caching via the CachedUseMacro property. After the initial call to this method, subsequent calls will return the cached instance rather than re-evaluating the use-macro expression repeatedly. This overload provides a mechanism to bypass that cache, via the bypassCache parameter.

Parameters:
contextA Tales.TalesContext to use for the discovery of the macro.
