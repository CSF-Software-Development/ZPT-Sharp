object CraigFowler.Web.ZPT.Tales.TalesContext.GetRootObject ( string  identifier)

Gets a reference to the object that serves as the 'start-point' of a path reference from an identifier string.

The following priority order is used to search for the requested reference:

  1. If it is the special string CONTEXTS then return the special root contexts dictionary.
  2. If the identifier can be found in the definitions list then return the relevant object.
  3. If the identifier can be found within the root contexts dictionary then return the relevant item from there.
  4. If the identifier has not been found then set "found" to false and return null.
Parameters:
identifierA System.String - the starting object name in a TALES 'path' string.
Returns:
A System.Object - the object reference to the start point of the path. Note that this may be null and that the parameter found should be checked in order to determine whether the object exists but was legitimately set to null, or whether it does not exist.
Exceptions:
ArgumentNullExceptionIf the parameter identifier is null.
ArgumentOutOfRangeExceptionIf the parameter identifier is an empty string.
RootObjectNotFoundExceptionIf the reference requested by the parameter identifier cannot be found in this context.
