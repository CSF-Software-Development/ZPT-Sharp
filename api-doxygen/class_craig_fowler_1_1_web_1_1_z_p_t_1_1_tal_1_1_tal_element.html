void CraigFowler.Web.ZPT.Tal.TalElement.Render ( XmlWriter  writer)

Overloaded. Renders the output of this TAL element to the given XmlWriter.

The order of operation for TAL directives is listed below.

Note that I am using the TAL 1.4.1 'proposed' ordering method. That is, processing omit-tag before attributes. For more information about this, see the comment at the bottom of http://wiki.zope.org/ZPT/TALSpecification14 and also the page https://bugs.launchpad.net/zope.tal/+bug/430662.

  1. define
  2. condition
  3. repeat
  4. content or replace
  5. omit-tag
  6. attributes
