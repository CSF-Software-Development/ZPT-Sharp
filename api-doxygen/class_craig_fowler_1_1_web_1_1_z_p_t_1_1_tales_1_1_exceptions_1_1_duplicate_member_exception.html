Detailed Description

Represents a TalesException that was raised due to encountering an ambiguous/duplicate member during the traversal of a Expressions.PathExpression.

This can be raised because either two members in the same type have been decorated with TalesAliasAttribute with identical values for the Alias property.

It could also be raised because no members within the target type could be found with aliases matchig the target identifier, and the identifier resolves to an overloaded member. Overloaded members are not supported.
