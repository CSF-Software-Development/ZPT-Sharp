Introduction to METAL and macros

METAL exposes the functionality of ZPT macros. Macros are reusable pieces of markup, which may be defined in the same or a different source file to the page template being rendered.

Macros are useful for defining and referencing common markup structures in order to maintain consistency. By referencing the macro rather than copying the markup directly, all usages may be updated by editing the source macro. This could be used for purposes such as:

  • A common page header/footer/navigation menu
  • An info-box about an item - such as a product
  • The boilerplate markup for the pages in a site/application

METAL attributes

The following attributes exist in the METAL namespace, activated using the metal: attribute prefix.

Attribute Summary
define-macro Define a METAL macro consisting of the current element and all of its children, providing a macro name
use-macro Replace the current element and all of its children/content with the specified macro
define-slot Define a slot within a macro, which acts as an optional extension point when the macro is used
fill-slot When using a macro, replace the named slot with the specified markup
extend-macro Used alongside define-macro to create a new macro based upon an existing one, extending and/or redefining its slots

Macros for the <html> element

It is easy to see from the examples above how macros may be used to provide individual elements of a page, much like ASP.NET user controls. What is less immediately obvious is the power of creating a macro using metal:define-macro attribute on the document root (such as the <html> element).

This manner of usage enables you to specify boilerplate markup in a "page boilerplate" macro. This could include common elements within the <head> element, for example, as well as a shared page header, footer & navigation as applicable. The "page boilerplate" macro then exposes slots for the actual page content. Pages within the site/application then may use this macro and provide their actual content via slot-fillers.
