ZPT-Sharp by craigfowler

Implementation of Zope Page Templates (ZPT) in C#, for use as a .NET MVC v2 ViewEngine

At the moment this page is little more than a placeholder for documentation of the project. The long-term intention is to create the documentation using ZPT-Sharp (using some kind of offline site-assembling tool) that turns a directory of ZPT templates into a static HTML website. See: https://github.com/craigfowler/ZPT-Sharp/issues/11 and https://github.com/craigfowler/ZPT-Sharp/issues/12
