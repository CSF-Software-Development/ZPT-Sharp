Detailed Description

Wrapper for a traversal delegate.

The rationale for this type, and for using delegates to traverse properties, methods, indexers and fields, is that it offers a significant speed/performance boost over just using MethodInfo, PropertyInfo and FieldInfo.

The raw reflection items may be converted into delegate instances and cached within the ObjectTraverser, which will significantly speed up all future calls.
