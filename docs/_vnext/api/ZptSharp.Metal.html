Namespace ZptSharp.Metal

Classes

AddDefinedMacroToGlobalScopeProcessorDecorator

Decorator for the ZptSharp.Rendering.IProcessesExpressionContext service which detects element nodes which have 'define-macro' attributes upon them. If such an attribute is found then the macro is stored in the current context's global scope for later usage.

MacroExpander

Implementation of ZptSharp.Metal.IExpandsMacro which first extends the macro and then fills its slots.

MacroProvider

Implementation of ZptSharp.Metal.IGetsMacro which uses an expression evaluator to get macros from attributes.

MacroUsageContextProcessor

A ZptSharp.Rendering.IProcessesExpressionContext which applies the METAL rendering process. This begins with detection of the metal:use-macro attribute.

MetalAttributeSpecProvider

Implementation of ZptSharp.Metal.IGetsMetalAttributeSpecs which uses string constants and a ZptSharp.Dom.IGetsWellKnownNamespace.

MetalContextProcessorFactory

Implementation of ZptSharp.Metal.IGetsMetalContextProcessor which gets the expression-context processor for METAL.

MetalDocumentAdapter

An adapter for an ZptSharp.Dom.IDocument which allows it to be used as ZptSharp.Metal.IProvidesMacros and ZptSharp.Expressions.IGetsNamedTalesValue.

MetalDocumentAdapterFactory

Implementation of IGetsMetalDocumentAdapter which uses dependency injection to backfill the other dependencies, as well as the document.

SlotFiller

Implementation of ZptSharp.Metal.IFillsSlots which replaces DOM nodes and removes the slot filler from the macro context when it is used.

SlotFinder

Default implementation of ZptSharp.Metal.IGetsSlots which gets a collection of slot-fillers for an node.

Interfaces

IGetsMetalDocumentAdapter

An object which gets an instance of MetalDocumentAdapter for a specified document.
