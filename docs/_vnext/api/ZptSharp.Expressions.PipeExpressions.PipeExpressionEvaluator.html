Remarks

A pipe expression is essentially an invocation of a System.Func`2 upon an existing variable. The function delegate's result/output type is always System.Object, although the input parameter may vary in type.

The result of the expression is the output of the function delegate, when executed against the specified variable. Pipe expressions are most useful for transforming or formatting values for output.
