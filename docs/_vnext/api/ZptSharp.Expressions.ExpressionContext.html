Remarks

The localDefinitions, repetitions & errorHandlers are shallow-copied by this constructor. This is because changes to these collections which are made inside of a context should not affect any other context

On the other hand, globalDefinitions is used directly without copying, because changes to the global definitions should affect other contexts, including those 'outside' of this one.
