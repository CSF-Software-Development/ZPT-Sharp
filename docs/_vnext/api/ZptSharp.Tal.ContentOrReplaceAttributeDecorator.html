Class ContentOrReplaceAttributeDecorator

Implementation of ZptSharp.Rendering.IHandlesProcessingError which handles both TAL 'content' or TAL 'replace' attributes. Their functionality is somewhat similar, and they cannot coexist on the same DOM node.
