Remarks

Please note that this class is provided for occasions when configuring ZptSharp within your own application's dependency injection configuration is not feasible. It is always best to fully integrate ZptSharp into your own DI config where you can.

The extension methods for IBuildsSelfHostingEnvironment are very similar to those for Microsoft.Extensions.DependencyInjection.IServiceCollection & System.IServiceProvider. Note though than when using this class, there is no equivalent for AddZptSharp(IServiceCollection). That is implied by the usage of this class. The extension methods which are avaialable are those which represent a meaningful choice or selection of an optional add-on.
