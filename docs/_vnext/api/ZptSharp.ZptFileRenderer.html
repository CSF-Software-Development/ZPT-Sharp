Class ZptFileRenderer

Implementation of ZptSharp.IRendersZptFile which uses an ZptSharp.IRendersZptDocument from a factory. It also uses an ZptSharp.Dom.IGetsDocumentReaderWriterForFile to attempt to get the appropriate document reader/writer for the file.
