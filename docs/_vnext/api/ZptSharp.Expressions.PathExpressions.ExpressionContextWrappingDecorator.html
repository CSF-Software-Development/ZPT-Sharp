Class ExpressionContextWrappingDecorator

A decorator class for the IGetsValueFromObject service, which detects instances of ZptSharp.Expressions.ExpressionContext and wraps them with an instance of NamedTalesValueForExpressionContextAdapter, so that subsequent classes in the decorator/chain may treat them as ZptSharp.Expressions.IGetsNamedTalesValue.
