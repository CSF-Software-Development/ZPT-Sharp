CreateTextNode(String)

Creates and returns a new text node from the specified content. Even if the content contains valid markup, it is strictly to be treated as text.

The node will be created in such a way as to be suitable for inclusion within the current node. That means that if the current node is CDATA, then the text will not be HTML encoded, otherwise it will.
