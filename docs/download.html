Download ZPT-Sharp

The following downloads are available for those who cannot or do not wish to use the ZPT-Sharp NuGet packages. NuGet is the the recommended way to use ZPT-Sharp in your own projects.

The ZIP downloads

The following ZIP downloads are for the most recent release: v1.0.0 (stable).

ZPT-Sharp complete package

The complete ZPT-Sharp download includes everything available. This is both the mandatory components (and their dependencies) as well as all of the supported plugins.

The ZIP file comes with its own README, which provides a summary of how to install the correct files for your use-case.

Download CSF.Zpt-complete.zip

Get the Source code

The recommended way to get the source code is to fork the repository on Github. For your convenience though, the following link will download a zip of the source code (provided by Github's archive feature).

Download v1.0.0.zip

Other versions

Links to previous releases and downloads are availalable on the project releases page for ZPT-Sharp, hosted on Github.
