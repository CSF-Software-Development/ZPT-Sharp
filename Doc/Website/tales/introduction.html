Introduction to TALES

TALES describes and evaluates expressions which describe the data model passed to your ZPT views; this could be a traditional data-model or an MVVM-style ViewModel.

Expressions are strings, which are included within the TAL or METAL attribute values at applicable points.

Expression prefixes

Each expression optionally begins with a prefix - a short identifier followed by a colon - indicating how it should be interpreted. Expressions which omit the prefix are treated as path expressions, which is the default expression-type. The available prefixes are:

  • path: expressions
  • string: expressions
  • not: expressions
