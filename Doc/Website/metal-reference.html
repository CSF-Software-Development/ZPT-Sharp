METAL reference

METAL stands for Macro Expansion for Template Attribute Language. It is the mechanism by which page templates may define and make use of macros, where macros are reusable areas of markup. If you are coming from an ASP.NET background, think of macros as a combination of master pages, and user controls.

define-macro

The first step in using METAL is to attach a metal:define-macro attribute to the root element which comprises the markup for the macro. Here is an example. 

<table>
  <tbody>
    <tr metal:define-macro="product_row">
      <td tal:content="product/Name">Name here</td>
      <td tal:content="product/Description">Description here</td>
      <td>
        <span tal:replace="product/Price/Value">9.99</span>
        <span tal:replace="product/Price/Currency">GBP</span>
      </td>
    </tr>
  </tbody>
</table>

In this example, the <tr> element is the root of the METAL macro. Wherever this macro is used, with a metal:use-macro attribute, that table row (and its children) will be spliced into the output document.
