ZPT-Sharp Nuget Packages

The following packages are yet to be released to the public Nuget repositories. Please check back once the project has gone to beta release.

The recommended way to install ZPT-Sharp into your own software project is via the Nuget package manager.

Because ZPT-Sharp is a customisable, plugin-based framework, there are two available strategies for installation. As well as packages for the individual components & plugins, fully-functional bundles are also available.

ZPT-Sharp Nuget bundles

For new users, those with simple application architectures, or those who want try ZPT-Sharp out, a bundle installation is recommended. The bundle packages include the relevant components and a pre-selected collection of plugins which will just get everything working.

Using a bundle, you should only need install a single package in order to get ZPT-Sharp up and running with your application. All bundles include a default selection of ZPT-Sharp plugins; two document providers and four expression evaluators:

  • HTML documents (HTML Agility Pack)
  • XML documents (XML Linq API)
  • String expression evaluator
  • Not expression evaluator
  • Path expression evaluator
  • CSharp expression evaluator

ZPT-Sharp for ASP.NET MVC5

This bundle includes the default plugins as well as the components required in order to use ZPT-Sharp as a ViewEngine for ASP.NET MVC5.

Install this bundle package into your web application project, make the required change to your view-engine registrations and you are ready to write views using ZPT.

ZptBuilder application

This bundle includes the default plugins and the ZptBuilder.exe command-line application. Choose this bundle to get a functional installation of the offline rendering application.

ZPT-Sharp API bundle

This bundle includes the default plugins and the ZPT-Sharp core logic components only. This package is intended for application developers who wish to make use of ZPT-Sharp as an API from their own software.

Hand-picking components & plugins

For advanced scenarios, particularly where you would like a non-default set of plugins installed, the components and plugins are available in separate packages. This installation strategy makes it possible to finely tailor what is installed. The trade-off is that you must select multiple packages in order to achieve a useful ZPT-Sharp installation.

CSF.Zpt.Abstractions

This is a minimal package containing abstract types and interfaces for ZPT-Sharp. It exposes no actual functionality and has no dependencies.

CSF.Zpt

This is the core logic of the ZPT-Sharp API, without any plugins or other content. It depends only upon CSF.Zpt.Abstractions.

CSF.Zpt.Cli

This is the logic of the command-line ZptBuilder application, without any plugins or other content. It depends only upon CSF.Zpt.

CSF.Zpt.MVC

This is the ASP.NET MVC5 ViewEngine, without any plugins or other content. It depends upon CSF.Zpt and the ASP.NET MVC components.

CSF.Zpt.DocumentProviders.HtmlHAP

This is the document provider plugin for rendering HTML documents. It depends upon CSF.Zpt and the HTML Agility Pack.

CSF.Zpt.DocumentProviders.XmlLinq

This is the document provider plugin for rendering XML documents using the newer XML/Linq API. It depends only upon CSF.Zpt.

CSF.Zpt.DocumentProviders.XmlLegacy

This is the document provider plugin for rendering XML documents using the older XML API. It depends only upon CSF.Zpt.

CSF.Zpt.ExpressionEvaluators.StringExpressions

This is the expression evaluator plugin for string expressions. It depends only upon CSF.Zpt.

CSF.Zpt.ExpressionEvaluators.PathExpressions

This is the expression evaluator plugin for path & local path expressions. It depends only upon CSF.Zpt.

CSF.Zpt.ExpressionEvaluators.NotExpressions

This is the expression evaluator plugin for not expressions. It depends only upon CSF.Zpt.

CSF.Zpt.ExpressionEvaluators.CSharpExpressions

This is the expression evaluator plugin for csharp expressions. It depends only upon CSF.Zpt.

CSF.Zpt.ExpressionEvaluators.PythonExpressions

This is the expression evaluator plugin for python expressions. It depends upon CSF.Zpt and IronPython.
