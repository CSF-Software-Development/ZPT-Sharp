Download ZPT-Sharp

Install via NuGet

The recommended way to use ZPT-Sharp in your own projects is to install it via the NuGet package manager. Please refer to the NuGet packages page for a list of the available packages.

Zip downloads

ZPT-Sharp is also available as a series of zip packages for those who do not wish to use NuGet. The following downloads are for the most recent release: v0.0.0 (release).

  • Complete package

    The complete package includes everything, both mandatory and optional components.

    Download CSF.Zpt-complete.zip

  • MVC 5 ViewEngine

    This zip file includes only the components required in order to use ZPT-Sharp as an ASP.NET MVC 5 ViewEngine.

    Download CSF.Zpt.MVC.zip

  • ZptBuilder.exe

    Only the command-line ZptBuilder.exe application and dependencies required to make it work.

    Download ZptBuilder.zip

  • API only

    For writing your own applications using ZPT-Sharp, this package includes only the core API components which you will need to reference from your own application logic.

    Download CSF.Zpt-api-only.zip

Get the Source code

The recommended way to get the source code is to fork the repository on Github. For your convenience though, the following link will download a zip of the source code (provided by Github's archive feature).

Download v0.0.0.zip

Other versions

Links to previous releases and downloads are availalable on the project releases page for ZPT-Sharp, hosted on Github.
