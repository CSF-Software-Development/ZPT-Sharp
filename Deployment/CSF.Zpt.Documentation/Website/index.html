Zen views for .NET with ZPT-Sharp

Overview

ZPT-Sharp is a .NET implementation of the Zope Page Templates specification. ZPT is an easy-to-understand syntax for writing HTML & XML pages with dynamic data. It may be used either with ASP.NET MVC, as a stand-alone offline application or as shared library.

ZPT is an attribute language, meaning that all of the statements and directives which control the rendering of a document are performed using attributes. This offers some distinct advantages over traditional page-rendering languages.

A sample ZPT document

Here is a small HTML document, which makes use of some of the ZPT TAL functionality, to bind model data to the HTML. 

<html xmlns:tal="http://xml.zope.org/namespaces/tal">
<head>
<title tal:content="string:Welcome to ${here/Name}!">Welcome to Placeholder Shop Name!</title>
</head>
<body>
<h1>You are visiting <span tal:replace="here/Name">Placeholder Shop Name</span></h1>
<p>Here are our products:</p>
<ul>
  <li class="product_listing" tal:repeat="product here/Products">
    <h2 tal:content="product/Name">Placeholder product name</h2>
    <div class="description" tal:content="structure product/RichHtmlDescription">
      <em>Placeholder product description</em> here, which might include <strong>HTML elements</strong>.
    </div>
    <strong class="price" tal:content="product/GetFormattedPrice">¥50.45</strong>
  </li>
</ul>
</body>
</html>
