Download ZPT-Sharp

For a complete list of releases and downloads, visit the releases page for ZPT-Sharp on Github.

The most recent stable release is v0.0.0, the following downloads are for that version.

ZPT-Sharp complete package

The complete package includes everything, both mandatory and optional components.

Download CSF.Zpt-complete.zip

ZPT-Sharp MVC 5 ViewEngine

This zip file includes only the components required in order to use ZPT-Sharp as an ASP.NET MVC 5 ViewEngine.

Download CSF.Zpt.MVC.zip

ZptBuilder command-line app

Only the ZptBuilder.exe application and dependencies required to make it work.

Download ZptBuilder.zip

ZPT-Sharp API only

For writing your own applications using ZPT-Sharp, this package includes only the core API components which you will need to reference from your own application logic.

Download CSF.Zpt-api-only.zip

Get the source code

The recommended way to get the source code is to fork the repository on Github. For your convenience though, the following link will download an archive of the source code for the latest stable version (provided by Github's archive feature).

Download v0.0.0.zip
