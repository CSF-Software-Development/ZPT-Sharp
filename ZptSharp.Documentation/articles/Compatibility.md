# ZptSharp compatibility
ZptSharp is written with compatiblity in mind. The NuGet packages use *multi-targeting* to
offer a wide variety of compatible framework versions.

| Framework      | Min version |
| -------------- | ----------- |
| .NET Framework | 4.6.1       |
| .NET Core      | 2.0         |
| .NET           | 5.0         |
| .NET Standard  | 2.0         |
