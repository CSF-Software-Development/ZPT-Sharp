ZPT-Sharp Page Builder Quick-start

Zope Page Templates can also help you build static HTML websites that require absolutely no server-side technologies to run. Write your pages using the METAL and TAL technologies and then make use of ZPT-Sharp's page builder to assemble those components as an HTML website.

What the ZPT-Sharp Page Builder does

The ZPT-Sharp Page Builder is a commandline utility that takes a directory structure containing ZPT documents, parses each document using the core ZPT functionality and then outputs the rendered results of those documents into an output directory as static HTML documents.

The main motivation to use the ZPT Page Builder is gaining access to the flexibility that METAL grants you in writing consistent pages, with reusable content regions, when the web hosting environment will not support any server-side technologies such as SSI. As an example, this documentation website was written 'offline' using ZPT and then assembled as a static HTML website using the Page Builder. The source for this documentation website is a good place to start if you would like to see an example of how a ZPT-Sharp Page Builder is structured.

Using the ZPT-Sharp Page Builder

Firstly, you will want a grounding in the syntax of ZPT pages, paying particular attention to METAL, which is the technology that allows use and reuse of content macros. TAL may also be useful, especially in combination with the options commandline parameter but METAL will be the bigger draw for Page Builder projects.

To assemble your directory structure of documents as a static HTML website, execute the page builder: CraigFowler.Web.ZPT.Builder.exe -o "output/path" -e ".html" "input/path" This will read all of the files within the input/path (with a filename matching the pattern *.pt) as ZPT documents. The page builder will render each of those documents, saving the output within output/path at the same location relative to the the input path, with the filename extension .html instead of .pt.

For further information, take a look at the full manual.
